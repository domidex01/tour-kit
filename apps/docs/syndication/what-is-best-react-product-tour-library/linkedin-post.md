React Joyride has 400K weekly downloads. It doesn't support React 19.

I benchmarked 7 React tour libraries on bundle size, accessibility, and React 19 compatibility. The two most popular options (React Joyride and Shepherd's React wrapper) are broken on React 19.

Most libraries also fail basic accessibility audits. Intro.js scored 7 axe-core violations. Missing ARIA labels, no focus trapping, buttons implemented as links.

If you're evaluating tour libraries for a React 19 project, the landscape is thinner than the "top 10" listicles suggest. I wrote up a comparison with actual benchmark data and a decision framework for choosing the right one.
