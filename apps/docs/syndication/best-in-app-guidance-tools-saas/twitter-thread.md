## Thread (6 tweets)

**1/** I tested 10 in-app guidance tools for SaaS. Pricing ranges from $69/month to $405,000/year.

Here's what I found 🧵

**2/** The market splits into 3 categories:

→ Headless libraries (Tour Kit, Shepherd.js): <8KB, full code control, free
→ Mid-market platforms (Userpilot, Appcues, Userflow): $240-$300/mo, no-code builders
→ Enterprise DAPs (Pendo, WalkMe): $48K-$405K/yr, multi-app workflows

**3/** Biggest surprise: no major vendor publicly benchmarks their bundle size impact.

SaaS platform scripts add 80-200KB of JavaScript. Tour Kit core? Under 8KB gzipped. Shepherd.js? 37KB.

If you care about Core Web Vitals on mobile, that gap matters.

**4/** The AI trend is real in 2026:

- Userflow's Smartflow generates flows from plain language
- CommandBar uses an AI search bar instead of traditional tours
- Pendo's guide builder suggests steps from user behavior

**5/** Quick decision framework:

React devs wanting control → headless library
Product team wanting speed → Appcues or Userpilot ($240-$300/mo)
Enterprise multi-app → Pendo or WalkMe ($50K+/yr)
Pre-Series A budget → UserGuiding ($69/mo)
