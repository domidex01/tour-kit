Can you ship production product tours without paying for a license?

We get this question constantly, so I wrote a full breakdown. Tour Kit's three core packages are MIT-licensed and free. The full tour engine, accessibility layer, router adapters, and headless components. Under 25KB gzipped. No watermark, no trial, no usage cap.

Pro ($99 one-time) adds analytics, checklists, announcements, surveys, and more. The math is simple: if the packages save you more than an hour of developer time, they've paid for themselves.

For context: Appcues costs $249/month ($8,964 over 3 years). AG Grid Enterprise runs $999+/dev/year. Tour Kit Pro is $99 total.

Full breakdown with package comparison tables and code examples: https://usertourkit.com/blog/tour-kit-free-vs-pro
