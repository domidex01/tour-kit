# Onboarding tools ranked by customer reviews (G2 + Capterra data)

Capterra lists 707 onboarding products with over 14,000 published reviews. Most are HR/employee onboarding tools. The product tour and in-app guidance category sits under G2's "Digital Adoption Platform" label.

We pulled G2 and Capterra scores for 10 tools that product and engineering teams actually evaluate, installed the open-source libraries, tested SaaS platforms, and cross-referenced review patterns.

Disclosure: Tour Kit is our project. Every claim is verifiable against G2, Capterra, GitHub, and npm.

## Key gaps in review platform data

**Open-source tools are absent.** Shepherd.js (13K GitHub stars) and React Joyride (7.6K stars) have no G2 or Capterra profiles. Review platforms systematically exclude the tools developers install most.

**Accessibility isn't measured.** No onboarding tool on G2 or Capterra prominently mentions WCAG compliance, ARIA support, or keyboard navigation in reviews.

**Technical impact is ignored.** Bundle size, Core Web Vitals impact, and JavaScript payload don't appear in any review.

## Decision framework

- **Headless library:** React developers with design systems who want code ownership
- **Opinionated open-source:** Need a working tour in under an hour
- **SaaS platform:** Product teams creating tours without engineering ($69-500+/month)
- **Enterprise DAP:** Product analytics bundled with guidance (five-figure contracts)
