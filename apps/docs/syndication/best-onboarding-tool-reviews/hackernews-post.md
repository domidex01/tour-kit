## Title: Onboarding tools ranked by G2 and Capterra reviews (with the gaps review platforms miss)

## URL: https://usertourkit.com/blog/best-onboarding-tool-reviews

## Comment to post immediately after:

I pulled G2 and Capterra scores for 10 onboarding and product tour tools and compared them against npm/GitHub data for the open-source options. Three findings stood out:

1. Open-source libraries (Shepherd.js at 13K stars, React Joyride at 7.6K) have no G2/Capterra profiles. The platforms PMs use to evaluate tools completely miss what developers actually install.

2. Accessibility is absent from the entire category. I searched both platforms for WCAG, ARIA, or keyboard navigation mentions in onboarding tool reviews. Zero results.

3. Capterra lists 707 "onboarding" products, but most are HR/employee onboarding tools (Rippling, Deel, BambooHR). The product tour category sits under G2's "Digital Adoption Platform" label, which creates genuine confusion for buyers.

Disclosure: I built one of the tools on the list (Tour Kit), so I have obvious bias. But the G2/Capterra ratings, GitHub stars, npm download counts, and pricing data are all independently verifiable. I tried to give fair treatment to competitors, including acknowledging where they're better.

Curious if anyone has experience with the G2/Capterra review process from the vendor side. The incentive structure seems to favor commercial SaaS tools over developer libraries in ways that aren't obvious to buyers.
