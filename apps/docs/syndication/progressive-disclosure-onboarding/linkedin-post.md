47% of users skip product tours entirely.

Of those who don't skip, linear tours (the "here's 10 steps on first login" approach) average just 53% completion. That's across 15 million interactions in Chameleon's benchmark dataset.

Progressive disclosure — revealing features based on behavior signals rather than dumping everything at once — pushes completion to 75%. And more importantly, paid conversions increase by 30%.

The framework that works:

Layer 1: One guided action on first session. Not five. One.
Layer 2: Contextual hints when users reach new screens (behavior-triggered, not calendar-based).
Layer 3: Power user features surface only after proven proficiency.

Jakob Nielsen introduced progressive disclosure in 1995. Google's Agent Developer Kit now uses the same principle for AI context management. The pattern keeps finding new applications because it maps to how humans actually learn.
