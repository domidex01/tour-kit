Web-only onboarding: $0-69/mo
Native mobile SDK: $299-999/mo
Enterprise: $48K+/year

There's almost nothing in between.

WCAG 2.2 now explicitly covers mobile with 9 new success criteria (touch targets, reflow, orientation).

Not one major onboarding tool documents WCAG compliance for mobile. Zero.

**5/** Plotline has the broadest coverage at 7 platforms but starts at $999/mo.
