"Mobile support" in onboarding tools means very different things depending on who's selling it.

I compared 8 onboarding tools for cross-platform coverage. Only 4 ship actual native mobile SDKs. The rest rely on responsive web design or let you toggle mobile tours off entirely.

The pricing gap is striking: web-only tools cost $0-69/month. Adding native mobile SDK support jumps to $299-999/month. Enterprise tools run $48K+/year.

One finding I didn't expect: WCAG 2.2 now covers mobile with nine new success criteria, but none of the major onboarding tools publicly document mobile accessibility compliance. For teams building products that serve diverse users, this is a real gap.

If you're evaluating cross-platform onboarding, the full comparison breaks down platform support, pricing, and accessibility for all 8 tools: https://usertourkit.com/blog/best-onboarding-tools-mobile-web
