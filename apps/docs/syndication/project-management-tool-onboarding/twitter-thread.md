**2/** The "Task Zero" concept:

Every PM tool has one moment that proves it works. The first real task a user creates, assigns, and completes.

Not a tutorial step. Real work. Trello nails this — the onboarding IS a Trello board.

**3/** Two onboarding paths nobody talks about:

The admin sees empty states and needs setup help.
The team member invited 3 days later sees populated projects and needs workflow help.

Building one tour for both = neither gets the right experience.
