## Title: Comparing Appcues alternatives for React developers – bundle size, pricing, and accessibility

## URL: https://usertourkit.com/blog/best-appcues-alternatives-developers

## Comment to post immediately after:

I wrote this after going through the Appcues pricing discovery process and realizing $249/month for product tours felt steep for a bootstrapped project. Did a systematic comparison of seven alternatives.

The most interesting findings: React Joyride has 603K weekly npm downloads but can't ship React 19 support because of its class component architecture. Shepherd.js is technically good but the AGPL license is a dealbreaker for most commercial apps. Driver.js is 5 KB but manipulates the DOM directly, which fights React's model.

I built Tour Kit (one of the tools compared), so I'm obviously biased. I tried to be upfront about that and honest about limitations — no visual builder, no mobile SDK, smaller community. The comparison table has verifiable numbers from npm, GitHub, and bundlephobia.

Curious what others are using for in-app onboarding. Most of the SaaS options seem to converge on $250+/month with third-party script injection.
