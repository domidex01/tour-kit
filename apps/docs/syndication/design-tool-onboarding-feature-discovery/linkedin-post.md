Design tools have 200+ features in one screen.

Figma: vector editing, auto layout, dev mode, prototyping, variables, components. Canva: AI image gen, brand kit, animations. All behind context menus and keyboard shortcuts users may never discover.

I compared how 5 major design platforms solve this:

- Figma uses opt-in 4-5 card tours (unchanged since 2022)
- Canva triggers 1-3 contextual cards based on user actions
- Adobe XD forced a 6-8 card linear walkthrough
- Sketch has zero in-app guidance
- Penpot offers different paths for designers vs developers

The surprising gap: nobody combines opt-in short tours + behavior-triggered progressive disclosure + full WCAG 2.1 AA accessibility.

A GitHub/Microsoft/DX study found developers with intuitive tools feel 50% more capable of creative problem-solving. With Figma at $15/editor/month, users who only discover 20% of features are paying full price for a fraction of the value.
