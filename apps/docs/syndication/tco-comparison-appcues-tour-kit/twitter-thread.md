**1/** I modeled 3 years of Appcues costs vs an open-source React alternative at three MAU tiers.

The gap ranges from $28K to $228K.

Every number is sourced. Thread with the key findings:

**2/** At 5,000 MAU:
- Appcues: $53,940
- Tour Kit (OSS): $25,599
- Savings: 53%

Includes all engineering time at $150/hr for the open-source path. Appcues includes seats, price increases, and MAU scaling.

**3/** At 25,000 MAU:
- Appcues: $114,000
- Tour Kit: $25,599
- Savings: 78%

Tour Kit's cost stays flat. Client-side libraries don't care how many users load the page. Appcues scales against your growth.

**4/** At 100,000 MAU:
- Appcues: $264,000+
- Tour Kit: $36,099
- Savings: 86%

That $228K difference is a full-time engineer for 18 months.

**5/** Where Appcues genuinely wins:

- Zero React developers on team
- Need onboarding live by Friday
- Under 2,500 MAU permanently
- PMs need to author flows without code

An open-source library isn't always the right call.
