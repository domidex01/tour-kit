Intro.js is the most starred product tour library on GitHub (23K stars). But its React wrapper hasn't been updated in three years.

For teams building with React 19, Tailwind, and modern design systems, the friction points add up: DOM manipulation that fights React's rendering model, AGPL licensing that requires source disclosure for commercial use, and accessibility gaps that need manual fixes.

I wrote a step-by-step migration guide covering the code changes, callback patterns, and the AGPL question. The comparison table shows bundle size (16.5KB vs <8KB), licensing, TypeScript support, and accessibility features side by side.

Full guide: https://usertourkit.com/blog/replace-intro-js-react
