# How to replace Intro.js with a modern React tour library

Intro.js has 23,800 GitHub stars and 215K weekly npm downloads as of April 2026. It earned that popularity. But its React wrapper hasn't been updated in over three years, its AGPL license forces source disclosure for commercial apps, and its DOM-centric architecture fights React at every turn.

This guide walks through replacing an Intro.js tour with Tour Kit, a headless React library that ships under 8KB gzipped (core) and uses MIT licensing. We cover step definitions, tooltip rendering, callbacks, accessibility, and the AGPL license question.
