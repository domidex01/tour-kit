# How to replace Intro.js with a modern React tour library

*A step-by-step migration guide for React developers*

*Originally published at [usertourkit.com](https://usertourkit.com/blog/replace-intro-js-react)*

Intro.js has 23,800 GitHub stars and 215K weekly npm downloads. It earned that popularity. But if you're building with React 19, Tailwind, and modern tooling, Intro.js creates friction in ways that are hard to patch around.

The React wrapper hasn't been updated in three years. The AGPL license requires source disclosure for commercial apps. And the DOM manipulation approach fights React's component model at every turn.

This guide covers replacing Intro.js with Tour Kit, a headless React tour library under 8KB gzipped with MIT licensing. The migration touches five areas: step definitions, tour lifecycle, tooltip rendering, callbacks, and persistence.

Read the full tutorial with code examples and troubleshooting: [usertourkit.com/blog/replace-intro-js-react](https://usertourkit.com/blog/replace-intro-js-react)
