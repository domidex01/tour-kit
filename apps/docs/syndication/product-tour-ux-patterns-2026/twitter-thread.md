**2/** The #1 finding: trigger type matters more than content.

**3/** Tour length is ruthless:

- 3 steps: ~72% completion
- 5+ steps: 80% skip rate

The fix isn't "write better steps." It's chunked micro-tours: three 3-step tours per feature instead of one 10-step marathon.

George Miller's 1956 working memory research applies directly.

**4/** Progress bars add +22% to completion. Checklists add +21% on associated tours. Milestone celebrations increase progression by 40%.

Small UX additions, big measurable impact.
