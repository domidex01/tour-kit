# Onboarding for developer tools: CLI, dashboard, and API

Most developer tools ship three surfaces: a CLI, a web dashboard, and an API. Most onboarding strategies only cover one.

Twilio's redesigned onboarding delivered a 62% improvement in first-message activation and a 33% improvement in production launches within 7 days. That happened because they stopped treating each surface as a separate product.

This guide covers onboarding patterns for all three surfaces with React code examples and activation metrics.
