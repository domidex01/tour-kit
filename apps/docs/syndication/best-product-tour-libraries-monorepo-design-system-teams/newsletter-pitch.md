# Newsletter Pitches

## This Week in React / Sebastien Lorber

**Subject:** Listicle: Product tour libraries ranked for monorepo/design system teams

Hi Sebastien,

We published a comparison of 7 product tour libraries tested inside a Turborepo monorepo with a shared Radix-based design system. The angle is specific to monorepo teams: style isolation, tree-shaking across workspace boundaries, peer dep hoisting, TypeScript at package boundaries.

Main finding: libraries that inject global CSS (Shepherd.js, React Joyride) create permanent style override debt. Headless libraries that ship zero CSS integrate cleanly.

Link: https://usertourkit.com/blog/best-product-tour-libraries-monorepo-design-system-teams

Disclosure: We built one of the listed libraries (userTourKit). Every claim is verifiable.
