Most PLG content is written by companies selling PLG tools.

The advice always ends the same way: "buy our $300/month product."

I pulled real pricing data and did the 2-year cost comparison that vendors skip:

Appcues path (2,500 MAUs): $8,000-$12,000
Open-source path: ~$3,168 (including engineering time)

The more interesting finding: Bessemer's research shows 68% of developers abandon trials because of setup friction. Only 12% cite pricing.

Which means the adoption problem is product design, not which onboarding SaaS you buy.

PLG companies grow 50% YoY vs 21% for traditional SaaS (OpenView Partners). That growth comes from time-to-value and self-service design, not tooltip placement.

For bootstrapped and engineering-led teams, a complete PLG stack now costs $0-$100/month:
- Open-source tour library for onboarding
- PostHog free tier for analytics + session replay
- GA4 for traffic tracking

Paid tools still make sense for non-technical teams, enterprise scale (50K+ MAUs), or mobile SDK requirements. But for most early-stage companies, the expensive SaaS tool isn't the only path to PLG anymore.
