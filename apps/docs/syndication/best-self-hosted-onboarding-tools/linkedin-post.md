Meta: EUR 1.2 billion. Uber: EUR 290 million. TikTok: EUR 530 million. All for transferring EU user data to the US.

Your onboarding tool sends behavioral data (step completions, click coordinates, feature adoption metrics) to third-party servers by default. With DORA, NIS2, and the EU Data Act all active in 2026, that's a growing liability.

I compared 8 self-hosted onboarding tools for teams that need to keep user data in their jurisdiction. The biggest insight: client-side JavaScript libraries are architecturally the strongest data sovereignty option because no user data ever hits a server. Nothing to transfer, nothing to audit.

The comparison covers libraries (Tour Kit, Shepherd.js, Driver.js, React Joyride, Intro.js) and Docker platforms (Usertour, Shepherd Pro, Guidefox) with licensing, accessibility, and a decision framework.

Full comparison: https://usertourkit.com/blog/best-self-hosted-onboarding-tools
