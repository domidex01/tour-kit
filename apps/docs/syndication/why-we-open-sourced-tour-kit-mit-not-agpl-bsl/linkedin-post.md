Every company that switched from open source to a restrictive license got forked.

HashiCorp → OpenTofu (5 days). Redis → Valkey (83% enterprise migration). Elastic reversed course entirely in 2024.

I built Tour Kit, a headless product tour library for React with 10 packages. When it came time to pick a license, the industry graveyard made the choice obvious: MIT.

Not AGPL (Google bans it company-wide, and it creates hidden friction for commercial users). Not BSL (100% fork rate, zero evidence of improved revenue).

The model that works: MIT core + paid extensions. Three packages are free forever. Eight are proprietary with a one-time fee. The boundary follows where value scales, not where we can extract maximum payment.

92% of open source projects now use MIT (2025 OSSRA). For UI libraries with zero switching costs, a restrictive license is just a reason to pick the alternative.
