# Product tours for EdTech: student engagement patterns that work

EdTech onboarding is broken. The average onboarding checklist completion rate for education platforms sits at 15.9%, below the 19.2% cross-industry average and far from the 24.5% that FinTech achieves (Userpilot, 2025). Most LMS platforms still rely on documentation dumps and training videos. Students don't read them. Teachers skip them.

This guide covers multi-persona tour routing, contextual hints, gamified checklists, WCAG accessibility compliance, and semester-aware fatigue prevention — all with working React code.
