Hey -- I put together a comparison of 9 open-source product tour libraries for React. Installed each one into a Vite 6 + React 19 project and compared bundle sizes (5KB to 30KB), TypeScript support, accessibility, and licensing.

Two things that surprised me: almost none of them claim WCAG 2.1 AA compliance, and Intro.js uses AGPL-3.0 which catches a lot of commercial teams off guard.
