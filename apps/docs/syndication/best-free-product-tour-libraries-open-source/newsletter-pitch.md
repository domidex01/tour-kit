## Email body (Cooperpress):

Hi,

I installed 9 open-source product tour libraries into the same Vite 6 + React 19 + TypeScript 5.7 project and compared them on bundle size, TypeScript support, accessibility, and licensing.

Key findings: bundle sizes range from 5KB (Driver.js) to 30KB (React Joyride). Intro.js uses AGPL-3.0, which catches commercial teams off guard. Almost no library in this space claims WCAG 2.1 AA compliance.

Every number is verifiable against npm and bundlephobia. There's a comparison table and decision guide at the end.

Link: https://usertourkit.com/blog/best-free-product-tour-libraries-open-source

Disclosure: I built one of the tools listed (User Tour Kit). Tried to keep it honest.

Thanks,
