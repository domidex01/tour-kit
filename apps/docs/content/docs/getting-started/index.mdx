# Getting Started

Learn how to install and configure TourKit in your React application.

## Overview

TourKit provides three packages:

1. **@tour-kit/core** - Headless hooks and utilities
2. **@tour-kit/react** - Pre-styled components (recommended)
3. **@tour-kit/hints** - Persistent hints system

## Which Package Should I Use?

- **Most users**: Start with `@tour-kit/react` - it includes everything you need
- **Custom UI**: Use `@tour-kit/core` for complete control over styling
- **Hints only**: Use `@tour-kit/hints` for persistent feature hints

## Requirements

- React 18 or 19
- Node.js 18+
