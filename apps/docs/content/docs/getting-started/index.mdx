# Getting Started

Tour Kit is a headless onboarding and product tour library for React. It's designed to work well with shadcn/ui and similar component libraries.

## Overview

Tour Kit provides a flexible foundation for building:

- **Product Tours** - Guide users through your application with step-by-step tours
- **Hints & Hotspots** - Highlight features with persistent hints
- **Announcements** - Display modals, banners, toasts, and more
