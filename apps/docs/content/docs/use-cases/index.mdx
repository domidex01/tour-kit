Tour Kit is a headless React library, which means it fits into whatever product pattern you're already running. These guides walk through the two most common shapes teams reach for first: **onboarding new users** and **announcing new features to existing users**.

## Not listed?

The same primitives — `Tour`, `TourStep`, `Hint`, and the `useTour` hook — also power contextual help, empty-state education, onboarding checklists, and role-based tours. Start from [Quick Start](/docs/getting-started/quick-start) and wire in the pieces you need.
