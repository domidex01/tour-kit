# AI Assistants

TourKit provides documentation optimized for AI coding assistants, LLMs, and automated tools.

## Available Resources

### llms.txt

A quick reference guide optimized for LLMs and AI coding assistants. Contains:

- Package overview and architecture
- Key exports for all packages
- Common usage patterns
- Type definitions

**Download:** [llms.txt](/llms.txt)

### llm-full.txt

Complete documentation dump for comprehensive context. Use this when you need the AI to have full knowledge of the library.

**Download:** [llm-full.txt](/llm-full.txt)

## Usage with AI Tools

### Claude Code

When working with Tour Kit in Claude Code, reference the llms.txt file:

```
Read the llms.txt file to understand the Tour Kit API
```

### Cursor / Copilot

Add llms.txt to your project context for better suggestions when working with Tour Kit components.

### Custom AI Integrations

Fetch the documentation programmatically:

```bash
# Quick reference
curl https://your-docs-site.com/llms.txt

# Full documentation
curl https://your-docs-site.com/llm-full.txt
```
