## Installation

```bash
pnpm add @tour-kit/react
```

## Components

- **Tour** - Main tour wrapper
- **TourStep** - Step definition
- **TourCard** - Tooltip card
- **TourOverlay** - Spotlight overlay
- **TourNavigation** - Next/prev buttons
- **TourProgress** - Progress indicator
