OVERVIEW
--------
In-app product announcements for React — modal, slideout, banner, toast, and spotlight variants with frequency rules, scheduling, audience targeting, and a priority queue for changelog and release communication

INSTALLATION
------------
    npm install @tour-kit/announcements
    # or
    pnpm add @tour-kit/announcements

Peer dependencies:
    react: ^18.0.0 || ^19.0.0

EXAMPLES

Example 1: Quick Start

    import { AnnouncementsProvider, AnnouncementModal } from '@tour-kit/announcements';
    
    function App() {
      return (
        <AnnouncementsProvider
          announcements={[
            {
              id: 'new-feature',
              variant: 'modal',
              priority: 'high',
              title: 'New Export Feature!',
              description: 'You can now export your data to CSV and PDF.',
              primaryAction: {
                label: 'Try It Now',
                onClick: () => console.log('Action clicked'),
              },
            },
          ]}
        >
          <AnnouncementModal id="new-feature" />
          <YourApp />
        </AnnouncementsProvider>
      );
    }
