OVERVIEW
--------
Feature adoption tracking, nudges, and dashboards for React — measure which product features users actually adopt and re-engage them with contextual prompts

INSTALLATION
------------
    npm install @tour-kit/adoption
    # or
    pnpm add @tour-kit/adoption

    export interface UseFeatureReturn {
      /** Feature definition */
      feature: Feature | null

      usage: FeatureUsage

      isAdopted: boolean

      status: AdoptionStatus

      useCount: number

    }

    export interface AdoptionStats {
      /** All features with their usage data */
