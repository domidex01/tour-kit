# Tour-Kit Documentation for LLMs

## Overview

Tour-Kit is a headless, accessible product tour library for React. It provides:
- Sequential guided tours with spotlight overlays
- Persistent hints for feature discovery
- Feature adoption tracking with nudges
- Interactive onboarding checklists
- Multi-variant product announcements
- Video/media embeds for tours
- Time-based scheduling for tours and announcements
- Analytics integration with popular platforms
- Full keyboard navigation and screen reader support
- SSR-compatible with Next.js and Vite

## Installation

```bash
npm install @tour-kit/react
# Optional packages
npm install @tour-kit/hints
npm install @tour-kit/adoption
npm install @tour-kit/analytics
npm install @tour-kit/announcements
npm install @tour-kit/checklists
npm install @tour-kit/media
npm install @tour-kit/scheduling
```

## Quick Start

```tsx
import { Tour, TourStep, TourCard, TourOverlay, useTour } from '@tour-kit/react';

function App() {
  return (
    <Tour id="welcome">
      <TourStep target="#feature" title="Welcome" content="Hello!" />
      <TourOverlay />
      <TourCard />
      <StartButton />
    </Tour>
  );
}

function StartButton() {
  const { start, isActive } = useTour();
  if (isActive) return null;
  return <button onClick={() => start()}>Start Tour</button>;
}
```
