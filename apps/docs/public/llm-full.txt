# Tour-Kit Comprehensive Documentation for LLMs

## Project Overview

Tour-Kit is a headless, accessible product tour and feature discovery library for React applications. It is designed for building user onboarding flows, product tours, and contextual hints.

### Architecture

```
@tour-kit/react ─────────┐
@tour-kit/hints ─────────┤
@tour-kit/adoption ──────┼──► @tour-kit/core
@tour-kit/announcements ─┤
@tour-kit/checklists ────┤
@tour-kit/analytics ─────┘
@tour-kit/media ─────────── (standalone)
@tour-kit/scheduling ────── (standalone)
```

- **@tour-kit/core**: Framework-agnostic hooks, utilities, and types
- **@tour-kit/react**: Pre-styled React components and headless variants
- **@tour-kit/hints**: Persistent hints for feature discovery
- **@tour-kit/adoption**: Feature adoption tracking with nudges
- **@tour-kit/analytics**: Plugin-based analytics integration
- **@tour-kit/announcements**: Multi-variant product announcements
- **@tour-kit/checklists**: Interactive onboarding checklists
- **@tour-kit/media**: Video/GIF/Lottie embeds for tours
- **@tour-kit/scheduling**: Time-based scheduling utilities

### Key Features

- Headless-first architecture with styled components
- Full TypeScript support with strict mode
- WCAG 2.1 AA accessibility compliance
- Keyboard navigation and focus management
- Screen reader announcements
- SSR-compatible (Next.js, Vite)
- Multi-page tour support with router adapters
- Persistence across sessions (localStorage, sessionStorage, cookies)
- Respects prefers-reduced-motion

## Basic Tour Example

```tsx
'use client';

import {
  Tour,
  TourStep,
  TourCard,
  TourCardHeader,
  TourCardContent,
  TourCardFooter,
  TourOverlay,
  TourProgress,
  TourNavigation,
  TourClose,
  useTour,
} from '@tour-kit/react';

export default function ProductTour() {
  return (
    <Tour id="welcome-tour">
      {/* Step 1 */}
      <TourStep
        target="#welcome-header"
        title="Welcome!"
        content="Let's take a quick tour of the key features."
        placement="bottom"
      />

      {/* Step 2 */}
      <TourStep
        target="#sidebar-nav"
        title="Navigation"
        content="Use the sidebar to navigate between sections."
        placement="right"
      />

      {/* Step 3 */}
      <TourStep
        target="#create-button"
        title="Create New"
        content="Click here to create a new project."
        placement="bottom-start"
      />

      {/* Spotlight overlay */}
      <TourOverlay />

      {/* Tour card */}
      <TourCard className="w-80">
        <TourCardHeader>
          <TourClose />
        </TourCardHeader>
        <TourCardContent />
        <TourCardFooter>
          <TourProgress variant="dots" />
          <TourNavigation />
        </TourCardFooter>
      </TourCard>

      {/* Your app content */}
      <AppContent />
      <StartTourButton />
    </Tour>
  );
}

function StartTourButton() {
  const { start, isActive } = useTour();

  if (isActive) return null;

  return (
    <button
      onClick={() => start()}
      className="fixed bottom-4 right-4 px-4 py-2 bg-primary text-white rounded-lg"
    >
      Start Tour
    </button>
  );
}
```

## Multi-Page Tour with Next.js

```tsx
'use client';

import {
  Tour,
  TourStep,
  TourCard,
  TourOverlay,
  MultiTourKitProvider,
  useNextAppRouter,
  useTour,
  usePersistence,
} from '@tour-kit/react';
import { useEffect, useState } from 'react';

export function OnboardingTour() {
  const router = useNextAppRouter();
  const persistence = usePersistence({ keyPrefix: 'myapp' });
  const [shouldShow, setShouldShow] = useState(false);

  useEffect(() => {
    const completed = persistence.getCompletedTours().includes('onboarding');
    setShouldShow(!completed);
  }, [persistence]);

  return (
    <MultiTourKitProvider
      router={router}
      routePersistence={{ enabled: true }}
      autoNavigate={true}
      onTourComplete={(tourId) => {
        persistence.markCompleted(tourId);
      }}
    >
      <Tour id="onboarding" autoStart={shouldShow}>
        {/* Home page */}
        <TourStep
          target="#hero"
          title="Welcome"
          content="Let's get started!"
          route="/"
          routeMatch="exact"
        />

        {/* Dashboard page */}
        <TourStep
          target="#dashboard"
          title="Dashboard"
          content="Your personal dashboard"
          route="/dashboard"
          routeMatch="startsWith"
        />

        {/* Settings page */}
        <TourStep
          target="#settings"
          title="Settings"
          content="Customize your experience"
          route="/settings"
        />

        <TourOverlay />
        <TourCard />
      </Tour>
    </MultiTourKitProvider>
  );
}
```

## Headless Custom UI

```tsx
import { Tour, TourStep, useTour } from '@tour-kit/react';
import { TourCardHeadless, TourOverlayHeadless } from '@tour-kit/react/headless';

function CustomTour() {
  return (
    <Tour id="custom-tour">
      <TourStep target="#feature" title="Custom" content="Fully custom UI" />

      {/* Custom overlay */}
      <TourOverlayHeadless
        render={({ overlayStyle, cutoutStyle, targetRect }) => (
          <div style={{ ...overlayStyle, background: 'rgba(0,0,0,0.7)' }}>
            {targetRect && <div style={cutoutStyle} />}
          </div>
        )}
      />

      {/* Custom card */}
      <TourCardHeadless
        render={({
          currentStep,
          currentStepIndex,
          totalSteps,
          next,
          prev,
          skip,
          floatingStyles,
          refs,
          isFirstStep,
          isLastStep,
        }) => (
          <div ref={refs.setFloating} style={floatingStyles} className="custom-card">
            <h2>{currentStep?.title}</h2>
            <p>{currentStep?.content}</p>
            <div className="flex gap-2">
              {!isFirstStep && <button onClick={prev}>Back</button>}
              <button onClick={isLastStep ? skip : next}>
                {isLastStep ? 'Finish' : 'Next'}
              </button>
            </div>
            <span>{currentStepIndex + 1} / {totalSteps}</span>
          </div>
        )}
      />
    </Tour>
  );
}
```

## Persistence Patterns

```tsx
import { usePersistence } from '@tour-kit/react';

function TourWithPersistence() {
  const persistence = usePersistence({
    storage: 'localStorage', // or 'sessionStorage', createCookieStorage()
    keyPrefix: 'myapp-tours',
  });

  // Check completed tours
  const completed = persistence.getCompletedTours();
  const wasSkipped = persistence.getSkippedTours().includes('onboarding');
  const dontShow = persistence.getDontShowAgain('onboarding');
  const lastStep = persistence.getLastStep('onboarding');

  // Mark tour state
  persistence.markCompleted('onboarding');
  persistence.markSkipped('onboarding');
  persistence.setDontShowAgain('onboarding', true);
  persistence.saveStep('onboarding', 2);

  // Reset
  persistence.reset('onboarding'); // Single tour
  persistence.reset(); // All tours
}
```

const features = [

      features={features}

## @tour-kit/analytics Full API

Plugin-based analytics for tracking tour, step, hint, and adoption events.

### Installation

```bash
npm install @tour-kit/analytics
```

### Setup

```tsx
import {
  AnalyticsProvider,
  createAnalytics,
  consolePlugin,
  posthogPlugin,
} from '@tour-kit/analytics';

const analytics = createAnalytics({
  plugins: [
    consolePlugin({ enabled: process.env.NODE_ENV === 'development' }),
    posthogPlugin(), // Uses window.posthog
  ],
  defaultProperties: {
    appVersion: '1.0.0',
  },
});

function App() {
  return (
    <AnalyticsProvider analytics={analytics}>
      {children}
    </AnalyticsProvider>
  );
}
```

### Hooks

```tsx
// Throws if analytics not available
const analytics = useAnalytics();

// Safe - returns null if not available
const analytics = useAnalyticsOptional();

// Usage
analytics?.track('custom_event', { key: 'value' });
analytics?.identify('user-123', { plan: 'pro' });
```

### TourAnalytics Methods

```tsx
const analytics = useAnalytics();

// Tour events
analytics.tourStarted('tour-id', 5);
analytics.tourCompleted('tour-id');
analytics.tourSkipped('tour-id', 2, 'step-3');
analytics.tourAbandoned('tour-id', 2, 'step-3');

// Step events
analytics.stepViewed('tour-id', 'step-1', 0, 5);
analytics.stepCompleted('tour-id', 'step-1', 0);
analytics.stepSkipped('tour-id', 'step-1', 0);
analytics.stepInteraction('tour-id', 'step-1', 'click');

// Hint events
analytics.hintShown('hint-id');
analytics.hintDismissed('hint-id');
analytics.hintClicked('hint-id');

// Generic tracking
analytics.track('custom_event', { data: 'value' });
analytics.identify('user-id', { email: 'user@example.com' });
await analytics.flush();
```

## Troubleshooting

### Tour not appearing
- Ensure target elements exist in DOM
- Check that TourOverlay and TourCard are inside Tour
- Use waitForTarget for dynamic elements

### Hydration errors (Next.js)
- Add 'use client' to tour components
- Or use dynamic import with ssr: false

### Multi-page tour not working
- Enable routePersistence in MultiTourKitProvider
- Use correct routeMatch mode
- Ensure router adapter is provided

### Styles missing
- Add @tour-kit packages to Tailwind content config
- Or import CSS directly
