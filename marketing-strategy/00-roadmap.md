# Tour-Kit Marketing Strategy Roadmap

> A complete go-to-market plan to launch tour-kit as the definitive open-source alternative to $250-900/mo SaaS onboarding platforms.

## How This Folder Works

Each numbered file is a standalone deliverable. Execute them in order — later documents build on earlier ones. When all are complete, you have a full marketing playbook ready for launch day.

| 05 | **Positioning & Messaging Framework** | One-liner, elevator pitch, value props by persona, key differentiators, objection handling, taglines | The single source of truth for how we talk about tour-kit everywhere. Every page, every post, every conversation draws from this. |

## Key Strategic Principles

### 1. Developer Authenticity Over Marketing Polish
Developers can smell marketing from a mile away. Every piece of content should sound like an engineer talking to another engineer. No superlatives ("best-in-class"), no buzzwords ("leverage"), no stock photos. Code samples, honest benchmarks, real limitations acknowledged.

### 2. Documentation IS Marketing
shadcn/ui, Tailwind, and Supabase all grew through exceptional docs. Your docs site at `apps/docs/` is your most important marketing channel. Interactive examples, copy-paste code, and a "working in 60 seconds" experience convert better than any ad.

### 3. Position Against Known Incumbents
Cal.com = "Open Source Calendly." Supabase = "Open Source Firebase." Tour-kit = **"Open Source Appcues"** or **"Headless React Joyride."** Every developer instantly understands the value when you position against something they already know and pay for.

### 4. Launch Multiple Times
Supabase launched on Product Hunt 16 times. Each extended package (surveys, segmentation, resource center, experiments) is a new launch moment. Don't blow everything on one day.

### 5. Free Must Be Genuinely Useful
The free tier (core, react, hints) must solve 80% of use cases for solo devs and small teams. If the free tier feels crippled, developers will choose a fully-free competitor and never come back. Generous free tiers build the adoption funnel that feeds Pro conversions.

### 6. Community Before Commerce
Build 1,000+ GitHub stars and genuine community usage before pushing the Pro tier hard. Tailwind sold $2M of Tailwind UI only after millions of free installs. The commercial product rides on the community's trust.

### 7. SEO Is a Compounding Asset
Every comparison page, tutorial, and migration guide compounds over time. A "Tour-Kit vs React Joyride" page written once will drive traffic for years. SaaS competitors (Chameleon, UserGuiding) already dominate these searches — we need to outrank them with more technical, more honest content.

### 8. One-Time Pricing Is the Wedge
$99 once vs $250-900/month is not a small difference — it's a category disruption. This is the single most powerful sales argument against every SaaS competitor. Lead with it everywhere.
