## 2. Market Drivers

### Product-Led Growth (PLG) Adoption
- PLG strategies achieve **64% activation rate** vs 25% SaaS baseline
- Trial-to-paid conversion: **15-25% for PLG** vs 5-10% sales-led
- Boosting activation rates by 25% increases revenue by 34%
- Onboarding dropoff rates: 30-50% for SaaS companies
- Users expect value within **5 minutes** of first touch

### React Ecosystem Dominance
- React holds **42.62% market share** among JS frameworks
- **85M weekly npm downloads**, 216K+ GitHub stars
- **11.2M websites** globally use React
- **80% of Fortune 500** companies use React in production
- Next.js has **52.9% adoption** among React developers

### shadcn/ui Convergence
- **110K+ GitHub stars**, 203K weekly npm downloads
- Becoming the default component approach for new React projects
- The "2026 indie stack" has converged: Next.js + TypeScript + Tailwind + shadcn/ui
- Tour-kit is built natively for this exact stack

### SaaS Cost Pressure
- Companies increasingly scrutinize recurring tool costs
- MAU-based pricing creates unpredictable cost scaling
- $3K-$48K/year for product tour SaaS feels indefensible to cost-conscious teams
- One-time pricing model gaining traction (Tailwind UI at $299, Intro.js at $9.99-$299)

**Gap:** No OSS library offers headless architecture + extended features (checklists, analytics, announcements) + shadcn/ui native + TypeScript strict + WCAG 2.1 AA. Tour-kit fills this gap entirely.

**Key insight:** SaaS competitors dominate every SEO keyword for product tours. **No OSS library has any meaningful content presence.** This is tour-kit's biggest opportunity.

### Tour-Kit's $99 Price Point Analysis
- **vs SaaS:** $99 once vs $2,988-$10,500/year = 97-99% savings. No-brainer for cost-conscious teams.
- **vs Build-it-yourself:** 40-80 hours of senior eng time ($6,400-$24,000) vs $99. Clear ROI.
- **vs Other OSS:** More feature-complete than any free option. Only $99 for 8 additional packages.
- **CAC ceiling:** Must stay below $25 for 4x return. Achievable via organic content + newsletter sponsorships.
- **No server costs:** Zero marginal cost per license = sustainable at any volume.
