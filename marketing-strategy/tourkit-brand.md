# TourKit Brand Identity

## Mission
Give React developers the most accessible, composable, and lightweight way to guide users through their products.

## Values
- **Technical excellence:** Strict TypeScript, tree-shakeable, sub-8KB core, WCAG 2.1 AA default
- **Developer-first:** Headless hooks ship separately from styled components. No forced opinions.
- **Transparency:** MIT open source, public roadmap, no hidden telemetry
- **Accessibility:** Focus management, keyboard nav, ARIA attributes, `prefers-reduced-motion` — built in, not bolted on

## Personality
- **Precise** — say exactly what we mean, no marketing fluff
- **Helpful** — write docs like the reader has 5 minutes to ship
- **Confident, not arrogant** — highlight real metrics, not superlatives
- **Craft-oriented** — care about gzip budget, keyboard traps, contrast ratios

## Visual Identity

| Element | Light Mode | Dark Mode |
|---------|-----------|-----------|
| Primary | Electric Blue `#0056ff` | Soft Blue `#5c9aff` |
| Secondary | Golden Amber `#a1790d` | `#c9a033` (decorative only) |
| Surface | `#fbfcfe` | `#151618` |
| Text | `#000023` | `#edeef2` |

- **Typography:** Geist Sans (body), Geist Mono (code)
- **Icons:** Lucide React, outline only, 2px stroke
- **Display name:** TourKit (two words, capital T and K)
- **Code name:** `tour-kit` (npm, GitHub, filesystem only)
- **Logo:** Needs creation. Placeholder: Lucide `Sparkles`

## Voice Rules
- Technical credibility first — every claim backed by code, numbers, or links
- Direct and concise — no filler words
- Honest about tradeoffs — say what we don't do
- Show, don't tell — 6-line code block > 3 paragraphs
- Never use: revolutionary, game-changing, best-in-class, seamless, robust, leverage, synergy, 10x, blazingly fast, delightful, empower, supercharge, unlock
- No emojis in docs, GitHub prose, or blog posts. Max 1 on Twitter if it adds meaning.

## Positioning
- **Primary:** "Headless onboarding for React."
- **Shorthand:** "The open-source Appcues."
- **Landing page:** "Your components. Your styles. Our engine."
- **Price wedge:** "$99 once. Not $300/month."
