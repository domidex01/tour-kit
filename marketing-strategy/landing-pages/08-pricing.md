## Section 4: FAQ

```
+------------------------------------------------------------------------+
|                                                                        |
|              Frequently asked questions.                               |
|                                                                        |
|   +------------------------------------------------------------------+|
|   |  Is Tour Kit really free?                                   [v] ||
|   |  Yes. MIT licensed. No usage limits, no feature gates, no       ||
|   |  tracking. The full library is free forever.                     ||
|   +------------------------------------------------------------------+|
|   |  What do sponsors get?                                      [v] ||
|   |  Priority support, private Discord, logo placement, and early   ||
|   |  access to new packages. The code itself is identical.           ||

|   |  Can I use Tour Kit commercially?                           [v] ||
|   |  Yes. MIT license permits commercial use with no restrictions.   ||
|   +------------------------------------------------------------------+|
|   |  What if I need a feature that doesn't exist?               [v] ||
|   |  Open a GitHub issue. Sponsors get feature prioritization.       ||
|   |  Enterprise sponsors can request custom development.             ||

|   |  How does this compare to no-code tools?                    [v] ||
|   |  No-code tools (Appcues, Userflow) are faster to set up but     ||
|   |  cost $200-500+/mo, can't match your design system, and own     ||
|   |  your data. Tour Kit is code-first, fully customizable, free.   ||
```
