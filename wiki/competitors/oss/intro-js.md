## Stats

| | |
|---|---|
| Stars | ~22,900 (highest in category, legacy) |
| Downloads | ~157K/week |
| License | AGPL-3.0 + Commercial ($9.99 – $299.99) |
| Bundle | ~10KB |
| Age | ~2013 |

## Strengths

- Highest star count (accumulated over ~12 years)
- Simple data-attribute API
- Lightweight
- Established brand recognition

## Weaknesses

- AGPL (see [Shepherd.js](shepherd-js.md))
- Not React-native — separate `intro.js-react` wrapper package
- Maintenance has slowed (8+ months between releases)
- DOM-based approach conflicts with React's virtual DOM
- Dated UI
- No extended features

## TourKit wins

- React-native hooks + components
- MIT (free tier)
- Headless
- TypeScript strict
- Modern composition patterns
- Active maintenance

## How to position against it

- Age isn't quality — "22K stars over 12 years ≠ best tool for React today"
- Show the wrapper friction (`intro.js-react` is a thin adapter)
- Modern TypeScript vs ~2013 JS patterns
