*Emerging OSS competitors positioning in the same space. Monitor closely.*

## OnboardJS

- **Position:** "Open-source headless onboarding framework for React and Next.js"
- **Threat level:** **Medium** — direct competitor on same keywords
- **Their angle:** Headless, React-first, wants the "open-source Appcues" slot
- **TourKit counter:** More packages (8 vs limited set), shadcn/ui native, shipping first. Move fast on SEO and comparison content.

## NextStepjs

- **Position:** Lightweight React/Next.js onboarding, zero deps beyond Motion
- **Threat level:** **Low** — limited scope, scope-competing with Onborda more than TourKit
- **TourKit counter:** Breadth (8 packages), not locked to a motion library

## Flows.sh

- **Position:** "You build UI, we handle orchestration" — product adoption platform
- **Threat level:** **Medium** — hybrid OSS/SaaS, different business model
- **TourKit counter:** No server dependency, code ships to `node_modules`, $99 one-time vs recurring

## Monitoring cadence

- Weekly: check star velocity on GitHub trending for React onboarding
- Monthly: audit their docs for positioning moves
- Quarterly: revisit the competitive landscape page and update here
