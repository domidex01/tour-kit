## Stats

| | |
|---|---|
| Stars | ~13,600 |
| Downloads | ~130K/week |
| License | AGPL-3.0 + Commercial ($50–$300) |
| Bundle | ~25KB |
| Category | OSS (framework-agnostic) |

## Strengths

- Framework-agnostic (React/Vue/Angular wrappers)
- High star count
- Active development
- Uses Floating UI for positioning (same as TourKit)

## Weaknesses

- **AGPL is a dealbreaker** for most commercial projects
- React wrapper is a thin adapter, not native
- Not headless
- Tours-only — no extended feature set
- Confusing tier structure (free AGPL / commercial / multiple commercial licenses)

## TourKit wins

- MIT license (free tier) + clean $99 one-time Pro
- Native React hooks, not a wrapper
- Headless architecture
- 8 packages, not just tours
- Clearer pricing

## How to position against it

- Lead with AGPL concern: "Your legal team doesn't want AGPL in a commercial product."
- Side-by-side: TourKit `useTour` hook vs Shepherd's imperative `new Shepherd.Tour()`
- React-native vs wrapper as a code-quality differentiator
