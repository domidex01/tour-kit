*Market size, TAM/SAM/SOM, timing, risks, and the feasibility verdict.*

> **Verdict: GO** — Strong market tailwinds, clear positioning gap, validated business model. March 2026 analysis.

## Market size & growth

### Product Tour Software Market
- 2024: **$550M** → 2033 projected: **$1.32B** (CAGR 10%)
- Source: Verified Market Reports

### Digital Adoption Platform (DAP) Market — broader category
- 2025: $1.18B–$1.9B (estimates vary)
- 2026: ~$1.59B (Fortune Business Insights)
- 2033 projected: $4.81B (CAGR 19.2%)
- Key players: WalkMe (SAP, $1.5B), Pendo ($300M revenue), Whatfix, Gainsight

### Digital Customer Onboarding Software — broadest category
- 2025: ~$8.4B, CAGR 22.3% through 2033

## Market drivers

### PLG adoption
- PLG achieves **64% activation** vs 25% SaaS baseline
- Trial-to-paid: 15–25% PLG vs 5–10% sales-led
- +25% activation ≈ +34% revenue
- SaaS onboarding dropoff: 30–50%
- Users expect value within **5 minutes** of first touch

### React ecosystem dominance
- React: **42.62% framework market share**, 85M weekly npm dl, 216K+ GitHub stars
- **11.2M websites** use React globally
- **80% of Fortune 500** run React in production
- Next.js: 52.9% adoption among React developers

### shadcn/ui convergence
- 110K+ stars, 203K weekly npm dl
- Default component approach for new React projects
- The "2026 indie stack" has converged: Next.js + TS + Tailwind + shadcn/ui
- TourKit is built natively for this exact stack

### SaaS cost pressure
- Companies scrutinizing recurring tool costs
- MAU-based pricing creates unpredictable scaling
- $3K–$48K/yr for product tour SaaS feels indefensible
- One-time pricing gaining traction (Tailwind UI $299, Intro.js $9.99–$299)

## Business model validation

### One-time license precedents

| Product | Price | Model | Signal |
|---|---|---|---|
| Tailwind UI | $299 once | Premium components for free framework | $2M+ revenue, funded Tailwind Labs |
| Intro.js | $9.99–$299 | AGPL + commercial tiers | Sustained since 2013 |
| Shepherd.js | $50–$300 | AGPL + commercial tiers | Active dev funded |
| Laravel Spark | $99 once | Premium starter kit | Sustained Laravel ecosystem |

### TourKit's $99 analysis

- **vs SaaS:** $99 once vs $2,988–$10,500/year = 97–99% savings. No-brainer for cost-conscious teams.
- **vs build-it-yourself:** 40–80 hours senior eng ($6,400–$24,000) vs $99. Clear ROI.
- **vs other OSS:** More feature-complete than any free option. $99 for 7 additional packages.
- **CAC ceiling:** Must stay below $25 for 4x return. Achievable via organic + newsletter sponsorships.
- **No server costs:** Zero marginal cost per license = sustainable at any volume.

## Timing — why now (2026)

| Factor | Opportunity |
|---|---|
| React 19 breaking changes | React Joyride (680K dl/wk) broken on React 19. Migration window is open NOW. |
| shadcn/ui convergence | Stack has converged. TourKit is the first onboarding library built natively for it. |
| SaaS cost fatigue | Companies cutting $5K–$50K/yr in SaaS spend. OSS alternatives trending. |
| PLG adoption accelerating | 64% activation for PLG vs 25% baseline. Every SaaS needs guided onboarding. |
| SEO vacuum | No OSS library ranks for any product tour keyword. First-mover advantage. |
| OnboardJS emerging | Direct competitor positioning as "open-source Appcues" — must move before they establish. |

## Risks

| Risk | Severity | Mitigation |
|---|---|---|
| React Joyride v3 stabilizes and regains trust | Medium | Ship superior DX, headless, and extended features they can't match |
| OnboardJS captures "open-source Appcues" position first | Medium | Move fast on content/SEO, launch with more packages (8 vs their limited set) |
| SaaS players build headless modes | Low | They won't cannibalize $300/mo revenue with headless |
| One-time pricing doesn't sustain development | Medium | Volume play + potential enterprise tier + consulting |
| Solo maintainer burnout | High | Build contributor community early, keep scope focused |

## Feasibility verdict

**GO.**

1. Clear market gap: no headless, TS-strict, accessible, shadcn-native tour library exists
2. Validated business model: one-time licensing works (Tailwind UI, Intro.js, Shepherd)
3. Perfect timing: React 19 migration + shadcn convergence + SaaS cost pressure
4. Massive SEO opportunity: zero OSS competition for product tour keywords
5. React ecosystem is 42% of web dev with 85M weekly npm dl — large enough
6. $99 = impulse corporate-card buy, no procurement

### Recommended first moves

1. Ship core + react + hints (free MIT tier)
2. Supabase-style launch week (HN + PH + Reddit + Twitter)
3. Publish comparison pages within 2 weeks of launch (SEO land grab)
4. Sponsor This Week in React newsletter ($430/issue, highest ROI paid channel)
5. Soft-launch Pro tier at week +4 post-launch
