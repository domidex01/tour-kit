*Free MIT tier + $99 one-time Pro. Why, what's included, how it compares.*

## Free tier

- MIT licensed
- Packages: `@tour-kit/core`, `@tour-kit/react`, `@tour-kit/hints`
- Covers 80% of use cases (product tours, styled components, hints/beacons)
- No tracking, no server, no gated features

## Pro tier — $99 one-time

- **Price:** $99 once. No subscription. No recurring fees.
- **Entitlement:** Lifetime updates
- **Scope:** 5 production sites
- **No:** per-seat pricing, no MAU limits
- **Delivery:** Code ships to `node_modules` — no server, no tracking, no vendor dependency
- **Packages included:** analytics, checklists, announcements, adoption, media, scheduling, surveys

See [product/packages.md](packages.md) for what each Pro package does.

## Why one-time vs subscription

| Angle | Position |
|---|---|
| Category disruption | "$99 once vs $300/month" is the wedge. Lead with it everywhere. |
| Developer trust | Code is yours forever. MIT-licensed free tier already covers the essentials. |
| Procurement-free | $99 = impulse corporate-card buy. No procurement. No renewal calls. |
| Zero marginal cost | No server, no CDN, no per-user billing. Sustainable at any volume. |

## One-time licensing precedents

- **Tailwind UI** — $299 one-time, $2M+ revenue, funded Tailwind Labs
- **Intro.js** — $9.99–$299 commercial license tiers, sustained since 2013
- **Shepherd.js** — $50–$300 commercial tiers
- **Laravel Spark** — $99 one-time
