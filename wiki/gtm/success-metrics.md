*Launch-week and 90-day targets.*

## Targets

| Metric | Launch Week | 90 Days |
|---|---|---|
| GitHub stars | 500+ | 2,000+ |
| npm weekly downloads | 1,000+ | 5,000+ |
| Docs monthly visitors | 5,000+ | 20,000+ |
| HN front page | Yes | — |
| Product Hunt top 5 | Yes | — |
| Pro licenses sold | 10+ | 100+ |
| Discord members | — | 200+ |
| Newsletter subscribers | 200+ | 1,000+ |
| SEO: page 1 for 3+ keywords | — | Yes |

## SOM Year 1

**$50K–$200K** — 500–2,000 Pro licenses at $99.

See [market/analysis.md](../market/analysis.md) for the full market sizing and business model validation.

## Leading indicators (weekly, not just launch-window)

- Issue response time < 24h
- PR response time < 48h
- npm weekly download trajectory (week-over-week)
- GitHub star velocity
- Docs site bounce rate
- Newsletter signup rate per docs visit

## What we track but don't optimize for

- Twitter follower count (vanity)
- Generic "impressions" / "reach"
- Email open rate (broken post-MPP)
